# homework-SimbirSoft
Домашние задания -
----------------
1)	Создать спринг бут проект с веб стартером. 
2)	Создать классы-dto репрезентующие 
Человека - ФИО(раздельно), дата рождения.
Книгу - Название, автор, жанр.
Статичный список людей и книг (штуки по 3) каждой сущности.
3)	Сделать 2 контроллера отвечающих за книги и людей.
4)	В каждом реализовать метод Get возвращающий 
а) Список всех книг и людей соответственно
б) Список всех книг по автору (пусть даже книга будет 1)
в) Список людей по имени 
5)	В каждом реализовать метод POST добавляющий новую книги или человека в список книг или людей соответственно и возвращающий новый дополненный список
6)	В каждом реализовать метод DELETE удаляющий книгу или человека из списка людей по ФИО и книг по Автору и названию. Возвращает ResponseEntity статус ОК. 
доп инфо. Допускается использование lombok. 
для вызова удобства вызова методов контроллера можете использовать POstman
